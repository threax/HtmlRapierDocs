Toggle Demo

This demo shows how to use the toggles.

Click Toggle Colors to make the red square turn blue.

 

How Does It Work?

The Markup

    <div data-hr-controller="toggleDemo">
        <button data-hr-on-click="toggleColors">Toggle Colors</button>
        <div data-hr-toggle="color" data-hr-style-on="background-color:blue" style="width: 100px; height: 100px; background-color: red;">
		&nbsp;
	</div>
    </div>

The markup defines a toggle named color with an on style that has a blue background and a standard style that is a red background. When the toggle is turned on the blue background will be added to its element style, which will override the red background. You can alternatively use classes with data-hr-class-on (or off).

The markup also defines a button that binds the toggleColors function to the click event of the button.

The Javascript 

"use strict";

jsns.run([
    "hr.controller"
], function (exports, module, controller) {
    function ToggleDemoController(bindings) {
        var toggle = bindings.getToggle('color');
        var on = false;
        
        function toggleColors(evt) {
            on = !on;
            if (on) {
                toggle.on();
            }
            else {
                toggle.off();                
            }
        }
        this.toggleColors = toggleColors;
    }

    controller.create("toggleDemo", ToggleDemoController);
});

The javscript defines a ToggleDemoController constructor function. This controller will get the toggle color from the bindings and it defines the toggleColors function, which toggles the color on and off. It is important to note that for the most part toggles do not maintain any state, so you have to track on and off status yourself. This demo is a bit contrived, usually you would put a series of toggles in a toggle group and activate them individually.
