Input Example

This example shows taking input and outputting it again.

 

Hello {{userName}} you have input some data with HtmlRapier.

 

Here input your name and press enter (or click submit). Your name will appear in the output section.

How Does It Work?

The Markup

<div class="row" data-hr-controller="inputDemo">
	<form class="col-md-6" data-hr-model="input" data-hr-on-submit="outputInfo">
		<div class="form-group">
			<label for="userName">Enter your name</label> 
			<input class="form-control" name="userName" type="text" />
		</div>
		<button type="submit">Submit</button>
	</form>
	<div class="col-md-6" data-hr-model="output">
		<p>Hello {{userName}} you have input some data with HtmlRapier.</p>
	</div>
</div>

 

Here we create a controller called inputDemo with 2 models named input and output. We also bind the function outputInfo from our controller object to the submit event on the form.

The input model will create a FormModel since it is on a form element.

The output model will create a TextNodeModel since it has variables, but no other special characteristics.

The Javascript

"use strict";

jsns.run([
    "hr.controller"
], function (exports, module, controller) {
    function InputController(bindings) {
        var inputModel = bindings.getModel("input");
        var outputModel = bindings.getModel("output");

        function outputInfo(evt) {
            evt.preventDefault();
            var data = inputModel.getData();
            outputModel.setData(data);
        }
        this.outputInfo = outputInfo;
    }

    controller.create("inputDemo", InputController);
});

 

Here we create our controller constructor function called InputController. It is bound to the inputDemo controller in the markup. Inside we gather up the input and output model and create a function called outputInfo that is added to the this object for the constructor. The outputInfo function will be bound to the form submit (using addEventListener(submit, func)). As such we get the event passed by the browser and prevent its default. Then we grab the data from input and put it into output.
